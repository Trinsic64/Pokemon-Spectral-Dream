# Maps

## Important compatibility note

The map assets in this folder can only be opened correctly using **AdAstra’s fork** of **Pokémon DS Map Studio**.

- Tool download (release): `https://github.com/AdAstra-LD/Pokemon-DS-Map-Studio/releases/tag/v2.2.1`
- Note from the fork: maps re-saved with that fork may be **incompatible** with vanilla Pokémon DS Map Studio.
